--- title: "CrowdSync" excerpt: "An automated album poster generator" collection: portfolio --- CrowdSync is a Flask-based web application designed to help users find and explore events such as concerts, sports, arts, theater, and comedy shows in their area. The project leverages the Ticketmaster API to fetch and display event information based on user inputs. (Python, Web Design, API Integration) https://crowdsync.pythonanywhere.com/
